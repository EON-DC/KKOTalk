### * 제작 예정 모듈은 총 다섯 가지
[Common] 공통 함수, 상수, String 저장하는 모듈입니다. common_module.py에 저장하고 사용합니다. 
<br>
[Code] 코드 쓰는 곳입니다.
<br>
[Client_Program] 클라이언트 런처 및 config 정보를 저장합니다.
<br>
[Server_Program] 서버 런처 및 config 정보를 저장합니다.
<br>
[Test] 테스트 하고싶으면 여기서 쓰십쇼
<br>
